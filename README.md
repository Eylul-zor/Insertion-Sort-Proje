# Insertion-Sort-Proje
PatikaDev Veri Yapıları ve Algoritmalar dersi kapsamında geliştirilen Insertion Sort projesi. <br>
Proje 1<br>
[22,27,16,2,18,6] -> Insertion Sort<br>
Yukarı verilen dizinin sort türüne göre aşamalarını yazınız.<br>
Çözüm:<br>
[22,27,16,2,18,6]<br>
[22,27,16,2,18,6]<br>
[16,22,27,2,18,6]<br>
[2,16,22,27,18,6]<br>
[2,16,18,22,27,6]<br>
[2,6,16,18,22,27]<br>
<br>
Big-O gösterimini yazınız.<br>
Çözüm:<br>
Average case: O(n^2)
Worst case: O(n^2)<br>
Best case: O(n)<br>

Time Complexity: Dizi sıralandıktan sonra 18 sayısı aşağıdaki case'lerden hangisinin kapsamına girer? Yazınız<br>
Çözüm:<br>
Average case: Aradığımız sayının ortada olması<br>
Worst case: Aradığımız sayının sonda olması<br>
Best case: Aradığımız sayının dizinin en başında olması.<br>

Dizi sıralandıktan sonra 18 sayısı ortalarda kaldığı için average case kapsamına girer.<br>
